# `listBaseTemplates`

List the premade base templates (light metadata). Clone one with createTemplate({ baseId }) or send from it directly via send({ templateId }).

**HTTP:** `GET /api/templates/base`

## Parameters

_None._

## Returns

`any`

## Example

```ruby
res = mk.listBaseTemplates()
```
